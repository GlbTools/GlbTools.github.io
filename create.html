Let’s Create Your Own GLB Site

Please log in with GitHub to start creating.

Step-by-Step Guide

Step 1: Fork Clone.Tools

Enter a unique name above (e.g., "rock.tools") and click "Fork Repo" to create your site.

Step 2: Check Fork Status

Status: Waiting to start...

Step 3: Create GLB Repo (Where we will save our .GLB Files)

Enter a name for your GLB storage repo and click "Create GLB Repo" to set it up.

Step 4: Check GLB Repo Status

Status: Waiting to start...

Step 5: Configure Your Site

Recommended: 800x450px, max 100KB

Step 6: Enable GitHub Pages

Go to your repo’s Settings > Pages, set the source to the main branch, and save.

Step 7: Upload Your GLBs

In "My Portal" on your new site, use the "Upload Model" section to add your .glb files, along with optional .txt and .png files.

Step 8: Share Your Creation

Once set up, select "Go Public" in My Portal to allow GLB.TOOLS to find your website.
