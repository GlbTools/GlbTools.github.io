Let’s Create Your Own GLB Site

Please log in with GitHub to start creating.

Step-by-Step Guide

Step 1: Fork Clone.Tools

Visit this link and click "Fork" to create your own copy.

Step 2: Name Your Site

Enter a name for your site above. Click "Create Repo" to set it up automatically.

Step 3: Configure Your Site

Use "My Portal" in the dropdown profile on your new website to configure it—it’s easier and built for this purpose.

Step 4: Enable GitHub Pages

Go to your repo’s Settings > Pages, set the source to the main branch, and save.

Step 5: Share Your Creation

Once set up, select "Go Public" in My Portal to allow GLB.TOOLS to find your website.
